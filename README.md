# Projeto Imersão Dev 10 - Alura & Gemini

## Descrição

- Estou participando da seleção dos melhores projetos da Imersão Dev com Alura e Google.

## Objetivo

- Desenvolver um projeto que seja capaz de buscar e exibir informações de linguagens de programação.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Git

## Como Usar

- Clone o repositório
- Abra o arquivo index.html no seu navegador
- Use o campo de busca para buscar por linguagens de programação
- Clique no botão de busca para buscar
- Clique no botão de saiba mais para ver mais informações sobre a linguagem de programação
